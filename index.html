Jogo Einstein

Pistas:

  1. O microsserviço verde é responsável por armazenar as informações detalhadas sobre cada produto e permite a criação de promoções e descontos para produtos específicos. Ele se comunica com o microsserviço vermelho para fornecer informações sobre o estoque disponível para cada produto.
  2. O microsserviço amarelo é responsável por processar os pagamentos dos clientes e garante a segurança dos dados dos clientes. Ele se comunica com o microsserviço vermelho para confirmar o pagamento do pedido e emitir um recibo automaticamente.
  3. O microsserviço azul é responsável por controlar a quantidade de produtos em estoque e alertar automaticamente quando um produto está em baixa no estoque. Ele se comunica com o microsserviço verde para obter informações detalhadas sobre cada produto e com o microsserviço vermelho para atualizar o status do pedido com base no estoque disponível.
  4. O microsserviço roxo é responsável por armazenar e gerenciar as informações dos usuários cadastrados na loja e permite a criação de uma lista de desejos personalizada para cada usuário. Ele se comunica com o microsserviço amarelo para processar o pagamento do pedido do usuário e com o microsserviço vermelho para fornecer informações sobre o status do pedido.
  5. O microsserviço verde é responsável por armazenar informações sobre os produtos disponíveis na loja e apresentar informações detalhadas sobre cada produto, como tamanho, cor e material. Ele se comunica com o microsserviço azul para atualizar as informações sobre o estoque disponível e com o microsserviço vermelho para fornecer informações sobre o status do pedido.
